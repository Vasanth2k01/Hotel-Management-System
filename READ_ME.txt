It is a " Hotel Management System " that I created in React JS. 

In this project, it includes Admin Page,Home Page, Room Booking Page, Payment wizard.

Description of this project : We have to preform operations that are related to frontend like view the home page, search date, checking the rooms availability and the status of the operation that user performed etc., And also it includes the admin UI.

Technologies:
	1. Html
	2. Css
	3. JavaScript
	4. Npm (For dependencies installation).
